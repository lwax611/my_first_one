# my_first_one
This is my entry point and the project shows an embarkment on a new journey
ENIAC a company specialized in selling Apple (tech) gadgets wants to enter the Brazilian market and is looking to partner with Magist which sells software as a service in Brazil
Magist provided 25 months of data for ENIAC to evaluate and see whether they are/ can be a suitable partner for them

Here I analyse the data and help make an educated decision on whether this makes sense
